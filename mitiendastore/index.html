Logotipo Primario

Cuando el logotipo se encuentra en un fondo blanco, preferentemente se debe utilizar la versión principal del logotipo.

El logotipo primario tiene dos usos alternos, dependiendo si se quiere utilizar en un fondo negro o en una sola tinta.

Logotipos Alternos

El logotipo tiene una variación horizontal para aprovechar mejor el espacio si se coloca en un formato horizontal. Se puede utilizar también el isotipo por sí solo o la tipografía en espacios pequeños.

Tipografías

Semplicita Pro

La tipografía institucional es Semplicita Pro. Se pueden utilizar todas las variantes de la tipografía para dar más énfasis a ciertas palabras o textos.
